# To-Do List FullStack Web application

A fullstack webapp to add and organize your task for each day to help you make a weekly schedule and keep you on track.
Just register with your email and you can easily do your tasks and mark them completed once you are done.

# Project Link

`https://to-do-list-main-pi.vercel.app/login`

# Technologies Used:
## Frontend
- NextJS
- ReactJS
- TailwindCSS
- Javascript
- CSS

## Backend
- Python and it's libraries
- MySQL
- FastAPI
